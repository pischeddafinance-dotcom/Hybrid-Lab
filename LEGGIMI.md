# Hybrid Lab — Percorso Formativo Operativo

Sito web del percorso formativo, pronto da pubblicare.

## File contenuti
- `index.html` — la pagina principale del percorso (le 7 fasi + Funnel)
- `manuale-funnel.html` — il Manuale del Funnel come pagina web
- `.nojekyll` — file tecnico, lascialo dov'è

---

## Come metterlo online con GitHub Pages (gratis, ~5 minuti)

### 1. Crea un account GitHub
Vai su https://github.com e registrati (se non ce l'hai già).

### 2. Crea un nuovo repository
- Clicca il "+" in alto a destra → **New repository**
- Nome: `hybrid-lab` (o quello che preferisci)
- Imposta su **Public**
- Clicca **Create repository**

### 3. Carica i file
- Nella pagina del repository clicca **uploading an existing file**
- Trascina TUTTI i file di questa cartella (`index.html`, `manuale-funnel.html`, `.nojekyll`)
- Clicca **Commit changes**

### 4. Attiva GitHub Pages
- Vai su **Settings** (in alto) → **Pages** (menu a sinistra)
- Sotto "Branch" seleziona **main** e cartella **/ (root)**
- Clicca **Save**

### 5. Fatto
Dopo 1-2 minuti il sito sarà online a:
`https://TUO-USERNAME.github.io/hybrid-lab/`

Condividi quel link con i tuoi studenti.

---

## Come modificarlo in futuro
1. Vai sul repository GitHub
2. Clicca sul file da modificare (es. `index.html`) → icona matita ✏️
3. Modifica e clicca **Commit changes**
4. Il sito si aggiorna da solo in 1-2 minuti

Ogni modifica resta salvata nella cronologia: puoi sempre tornare indietro.
